# -SOS-
『災害時の在宅避難者を支える、SOSピン打ちマップ』
## 福祉・地域連携の視点
在宅避難者の課題は、物資不足だけではなく、困りごとが周囲に伝わらず孤立してしまうことにある。
高齢者、障害者、医療的ケア児者など、支援を必要とする方々が安心して地域で生活を継続するためには、必要な情報や支援につながる仕組みが必要である。
また、災害時には本人だけでなく、家族、近隣住民、民生委員、ケアマネジャー、訪問看護師、地域包括支援センター、社会福祉協議会など、平時から関わりのある支援者との連携が重要となる。
このSOSマップは、単なる物資要請システムではなく、人と人をつなぎ、地域の支え合いを促進するシチズンコネクトの基盤となることを目指している。
災害時だけ使う仕組みは定着しにくいため、平時から見守りや地域情報共有にも活用できる仕組みとして育てることで、発災時にも自然に活用できることを目指す。
